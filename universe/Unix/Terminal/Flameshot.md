# Flameshot

Powerful screenshot tool with built-in annotation. Draw arrows, boxes, text, blur regions, and copy/save — all before saving the screenshot.

## Install

```bash
sudo apt install flameshot
```

## Usage

```bash
flameshot gui             # interactive screenshot
flameshot full            # full screen capture
flameshot screen          # capture current screen
flameshot gui -d 3000     # 3 second delay
flameshot config          # open settings
```

In the GUI overlay:

- Draw with pencil, arrows, rectangles, circles
- Add text, blur/pixelate regions
- Pin screenshot on screen
- Copy to clipboard or save to file
- Upload to Imgur (optional)

Tip: bind `flameshot gui` to `Print Screen` in your desktop settings.
